# 🧭 Navigation Guide

## How to Use the Sidebar

The sidebar is your main navigation tool in SyntaxFlow. Here's how to use it:

### Opening the Sidebar

The sidebar should be **open by default** when you launch the app. If it's closed:

1. **Look for the arrow icon (☰)** in the top-left corner
2. **Click the arrow** to expand the sidebar
3. The sidebar will slide out from the left

### What's in the Sidebar?

#### 1. App Header
- **SyntaxFlow logo** and title
- Quick description of the app

#### 2. Navigation Menu
You'll see 5 pages listed:
- **🏠 Home** - Landing page with demo
- **📊 Analytics** - Data visualization dashboard
- **🔬 Simplify** - Sentence simplification tool
- **🌳 Visualize** - Syntax tree viewer
- **📚 Dataset** - Data explorer

#### 3. Quick Tips
Helpful reminders and shortcuts

### Navigating Between Pages

**Method 1: Click Page Names**
- Simply click any page name in the sidebar
- The page will load instantly
- Current page is highlighted

**Method 2: Use Keyboard**
- Press `[` to open/close sidebar
- Use arrow keys to navigate (in some browsers)

### Sidebar Features

#### Always Accessible
- The sidebar is available on every page
- Your navigation is never blocked
- Quick access to all features

#### Responsive Design
- Automatically adapts to screen size
- Can be collapsed on mobile
- Smooth animations

#### Visual Feedback
- Current page is highlighted
- Hover effects on page names
- Smooth transitions

## Troubleshooting Navigation

### Sidebar Won't Open
**Problem:** Can't see or open sidebar

**Solutions:**
1. Look for ☰ icon in top-left
2. Refresh page (F5)
3. Clear browser cache
4. Try different browser
5. Check screen width (needs >768px)

### Pages Won't Load
**Problem:** Clicking page does nothing

**Solutions:**
1. Wait for current page to finish loading
2. Check browser console (F12) for errors
3. Refresh the app
4. Clear Streamlit cache
5. Restart the app

### Navigation is Slow
**Problem:** Pages take long to load

**Solutions:**
1. Clear Streamlit cache
2. Reduce data size
3. Close other tabs
4. Restart browser
5. Check system resources
